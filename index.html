I blog about web development. Here are some of my best entries:

  • About Fonts in SVG — I investigate how fonts are rendered in SVG and how SVG fonts can work as webfonts.
  • Multi-Lingual Fonts with Typotheque and Font Face — an investigation into rendering non-latin scripts using web fonts.
  • The Long Road from XHTML to HTML — an article on the rise and fall of XHTML.
  • Declaring Languages in HTML 5, going through the rabbit hole that is internationalization of HTML documents.
  • A bit of SVG and Canvas — SVG and Canvas have gained currency because of greater support in HTML 5. This post explains the differences between the two and how to use them with older browsers.
  • Using Data URIs in CSS — A post on how Data URIs can be used to render fonts and images.
  • Drop Shadows with CSS 3 – A simple way to create flexible drop shadows.
  • What Web Designers need to know about EPUB - An introduction to the EPUB book format for web designers and how to take advantage of it.
